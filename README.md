
# Color Clock
https://github.com/user-attachments/assets/e43cf6dd-86e3-4c80-bc69-e50c8e173ae1


A dynamic glassmorphism-inspired digital clock built with React, Vite, and date-fns.

The clock displays the current:
- Date
- Time
- Dynamic HEX background color based on the current time

The UI includes:
- Glassmorphism styling
- Animated gradients
- Responsive layout
- Real-time updates every second

---

## Tech Stack

- React
- Vite
- CSS3
- date-fns
